# GAMESTORE

GAMESTORE è una piattaforma web moderna progettata per facilitare la scoperta e l'acquisto di videogiochi. Il sito offre un'esperienza utente intuitiva con una vasta collezione di titoli, categorie organizzate e saldi settimanali attraenti.

## Caratteristiche Principali
- **Scoperta di Giochi**: Sezione dedicata ai giochi in evidenza con i titoli più giocati
- **Ricerca Avanzata**: Barra di ricerca centralizzata per trovare rapidamente i giochi desiderati
- **Categorie**: Esplora giochi suddivisi per genere (Azione, RPG, Sport, Trending)
- **Saldi Settimanali**: Offerte esclusive con sconti fino al 90%
- **Navigazione Intuitiva**: Menu principale con accesso a Store, Saldi e Categorie
- **Sistema di Accesso**: Pulsante Accedi per gli utenti registrati

## Tecnologie Utilizzate
- **HTML5**: Struttura semantica
- **CSS3**: Styling responsive e gradient moderni
- **JavaScript**: Navigazione dinamica

## Design
- **Tema Scuro**: Palette di colori scura (#0F1429, #242B42) con accenti ciano (#00D9FF)
- **Gradient Background**: Sfondo con gradiente verticale per un aspetto moderno
- **Layout Flessibile**: Utilizzo di Flexbox per layout responsive

## Come Utilizzare
1. Aprire `index.html` in un browser web
2. Esplorare i giochi in evidenza
3. Utilizzare la barra di ricerca per trovare titoli specifici
4. Navigare alle categorie o ai saldi tramite il menu principale
5. Cliccare su una categoria per visualizzare il catalogo completo
